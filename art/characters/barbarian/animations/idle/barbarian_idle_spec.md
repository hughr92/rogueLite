# Barbarian Idle Animation (V1)

## Frame Count
4 frames

## Loop Type
Ping-pong hybrid loop:
1 → 2 → 1 → 3 → 1 → 4 → repeat

## Frame Details

### Frame 1 (neutral)
- Base reference pose
- Balanced stance

### Frame 2 (compression)
- Slight downward movement
- Knees bent
- Shoulders lowered

### Frame 3 (expansion)
- Slight upward movement
- Chest expanded
- Shoulders raised

### Frame 4 (weight shift)
- Slight lean to one side
- Subtle torso rotation

## Key Constraints
- Maintain exact proportions from master sprite
- No change to silhouette readability
- Movements must be subtle but clearly readable at small scale

## Notes
- Animation should feel heavy and grounded
- Avoid floaty or exaggerated bounce
