# Skill Effect Placeholders

This folder stores placeholder PNGs for skill visual effects.

Naming convention:
- `<skill_id>_effect.png`

Layout:
- `class/<class_id>/...`
- `weapon/<weapon_id>/...`
- `general/...`

To scaffold missing placeholders for current skills, run:

```bash
node tools/create-skill-effect-placeholders.js
```
